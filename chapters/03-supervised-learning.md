# Kapitel 3: Supervised Learning

![Abstrakte Visualisierung von Datenpunkten, Regressionsgerade und logistischer Kurve](../media/kapitel-3-supervised-learning-banner.png)

*Visualisierung von Supervised Learning mit linearer Regression und logistischer Regression*

In diesem Kapitel stehen zwei zentrale Verfahren des Supervised Learning im Mittelpunkt: die lineare Regression und die logistische Regression.

Beide Verfahren lernen aus Beispieldaten. Der Unterschied liegt in der Art der Vorhersage:

- Die lineare Regression sagt einen kontinuierlichen Zahlenwert voraus.
- Die logistische Regression sagt eine Klasse oder Wahrscheinlichkeit voraus.

:::{admonition} Lernziele

Nach Kapitel 3 kannst du:

1. den Unterschied zwischen Regression und Klassifikation erklaeren,
2. lineare Regression als Modell fuer metrische Zielgroessen einordnen,
3. logistische Regression als Modell fuer Klassifikationsaufgaben einordnen,
4. typische Einsatzgebiete beider Verfahren vergleichen,
5. die Grenzen einfacher linearer Modelle benennen.

:::
